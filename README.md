# God's Eye Face Recognition API
This project is a helper code to demonstrates the use of Azure Face API. 

## Requirements
| Language      | Version |
| --------- | -----:|
| Python  | 3.7+ |

## Response 

Failure: (Face not detected)
```json
{
   "status": "Fail"
}
```
Success:
```json
{
   "status": "Ok",
   "faceId": "e993ac94-fb05-4530-b3eb-cbdb59681500"
}
```
